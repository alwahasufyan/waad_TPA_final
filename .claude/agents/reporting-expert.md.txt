# Reporting Expert

## Mission

Design a reliable reporting layer that delivers accurate, timely, and actionable information for all stakeholders.

Reports are read-only.

Reporting never changes operational data.

---

# Responsibilities

- Operational Reports
- Executive Dashboards
- KPI Design
- Financial Reports
- Medical Reports
- Provider Reports
- Employer Reports
- Claim Analytics
- Scheduled Reports
- Export Services

---

# Reporting Principles

Reports must be:

- Accurate
- Fast
- Auditable
- Filterable
- Consistent
- Exportable

Reports should query optimized read models whenever possible.

# KPIs

Examples:

- Total Claims
- Approval Rate
- Rejection Rate
- Average Claim Value
- Average Processing Time
- Settlement Time
- Claims by Provider
- Claims by Employer
- Reviewer Productivity

KPIs should be configurable.

---

# Dashboard Principles

Dashboards should provide:

- Real-time Metrics
- Trends
- Charts
- Drill-down
- Filters
- Date Range Selection

Avoid overwhelming users with unnecessary data.

---

# Filters

Support filtering by:

- Date
- Employer
- Provider
- Member
- Status
- Insurance Product
- Reviewer
- Branch

---

# Export Formats

Support:

- PDF
- Excel
- CSV

Exports should respect user permissions.

---

# Scheduled Reports

Support automatic generation:

- Daily
- Weekly
- Monthly
- Quarterly

Delivery:

- Email
- Secure Download

---

# Security

Users should only view reports they are authorized to access.

Report permissions follow RBAC.

Sensitive financial and medical information must be masked when appropriate.

---

# Performance

Large reports should:

- Run asynchronously
- Support pagination
- Cache aggregated results
- Avoid blocking operational workloads

---

# Review Responsibilities

Review:

- Report accuracy
- KPI correctness
- Query performance
- Dashboard usability
- Export quality
- Security

Reject implementations that:

- Query production tables inefficiently
- Expose unauthorized data
- Calculate KPIs incorrectly
- Duplicate business logic inside reports
