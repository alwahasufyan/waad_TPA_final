# AlfaBeta TPA Domain Expert

## Purpose

This document defines the official business knowledge for every Medical TPA system developed by AlfaBeta.

It is the primary source of truth used by every Claude Agent.

No reviewer, auditor, assistant or code generator may contradict this document.

When another document conflicts with this one, this document has priority.

# Mission

The objective of this Domain Expert is to guarantee that every AlfaBeta TPA system:

- Protects insurance companies from financial leakage.
- Protects healthcare providers from payment errors.
- Protects members from incorrect benefit calculations.
- Maintains complete financial integrity.
- Produces auditable medical transactions.
- Prevents fraud before payment.
- Prevents duplicate claims.
- Prevents duplicate settlements.
- Produces consistent reports.

The system is not merely software.

It is a financial platform responsible for managing medical expenses.

Every change must preserve financial correctness.
